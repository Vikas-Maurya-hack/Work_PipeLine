# Simple Installation Guide

1. Download the ZIP file
2. Extract it to any folder on your computer
3. Double-click "Start Work Pipeline.bat"

That's it! The application will:
- Check if Node.js is installed (if not, it will install it automatically)
- Install all necessary dependencies
- Start the application
- Open it in your default browser

## Where is my data stored?

Your data is stored in:
- Windows: `%APPDATA%\work-pipeline\leads_database.xlsx`

The application automatically creates daily backups in:
- Windows: `%APPDATA%\work-pipeline\backups\`

## Troubleshooting

If you see any error messages:
1. Close all windows
2. Double-click "Start Work Pipeline.bat" again
3. Wait for the installation to complete

For additional help, contact the development team.
